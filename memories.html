held onto anyway

Small things,
kept too carefully.

Not the big moments. The ordinary ones that decided to stay.

memory

The playlist

Twelve songs, made for a car ride that lasted eleven minutes. I still haven't deleted it.

memory

The handwriting

A note passed once, years ago. I could still pick that handwriting out of a hundred others.

memory

The umbrella

One umbrella, two people, neither of us in a hurry to get where we were going.

memory

The almost-call

Dialed, let it ring twice, hung up. Told myself it was a pocket-dial. It wasn't.

memory

The empty chair

Saved at every table, out of habit more than hope, for longer than I'd like to admit.

memory

The last message

Still unread on my end. Some doors are easier left slightly open.
